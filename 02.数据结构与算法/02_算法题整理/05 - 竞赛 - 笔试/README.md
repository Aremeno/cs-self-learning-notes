# 竞赛&笔试说明

## 文件夹用途

存放竞赛、笔试相关的资料和笔记，核心作用是「考前速记、针对性备考」，帮你应对笔试和竞赛，快速抓取高频考点。

## 内容规范

1. 一级分类：「高频考点速记」「笔试真题」「模拟赛」三个子文件夹；
    
2. 高频考点速记：单独建「高频考点速记.md」，内容包含：常用公式、复杂度对照表、常见坑点、考场代码小技巧（快读、endl替换等）；
    
3. 笔试真题：按公司/赛事建立子文件夹（例：字节跳动、美团、蓝桥杯），每道真题单独建笔记，内容包含：来源、题目简述、考点、思路、核心代码；
    
4. 模拟赛：按模拟赛日期建立子文件夹，记录模拟赛中的错题、难点、收获。
    
## 使用建议

- 考前1-2周，重点背诵「高频考点速记」，熟悉坑点和技巧；
    
- 笔试前，集中刷对应公司的真题，总结考点规律；
    
- 模拟赛后，及时整理错题，补充到对应题型和错题本文件夹，强化薄弱点。
