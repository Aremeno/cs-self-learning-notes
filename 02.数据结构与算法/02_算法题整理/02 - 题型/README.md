# 按题型分类说明

## 文件夹用途

核心文件夹，按算法/数据结构题型归类所有「值得记的题目笔记」，目的是「归类复盘、强化同类题思路」，帮助掌握题型规律，而非孤立记题。

## 内容规范

1. 一级分类：按题型建立子文件夹（例：双指针与滑动窗口、动态规划、图论）；
    
2. 单题笔记：每个值得记的题目单独建 .md 文件，文件名格式：「题号-题目名.md」（例：LeetCode55-跳跃游戏.md）；
    
3. 单题笔记需包含：题号&题目名、难度、题型标签、核心思路（1-3句）、关键步骤、核心代码片段、易错点、同类题题号；
    
4. 仅存放「经典思路题、可复用思路题」，简单水题、纯模拟题不放入此处。
    
## 使用建议

- 刷题后，将经典题、思路有价值的题，整理后放入对应题型文件夹；
    
- 复习时，按题型批量回顾（例：集中复习10道动态规划题），总结同类题规律；
    
- 同类题标注清楚，方便对比差异、加深理解。
