# 按平台刷题说明

## 文件夹用途

按刷题平台分类，记录「每日刷题流水账」，核心作用是「记录刷题进度、临时归档」，不做深度整理，方便后续筛选精华题目。

## 内容规范

1. 一级分类：按平台建立子文件夹（例：LeetCode、AcWing、洛谷）；
    
2. 二级分类（可选）：LeetCode可新增「热题100」「面试经典」「周赛」子文件夹，按刷题场景细分；
    
3. 每日记录：按日期建 .md 文件（例：2026-04-14刷题记录.md），内容包含：日期、平台、今日刷题列表（题号-题目名-难度-掌握情况）、今日总结（新学思路、错误、需整理的题目）；
    
4. 不做深度整理，仅记录当天刷题情况，避免占用过多时间。
    
## 使用建议

- 每天刷题结束后，10分钟内完成当日记录，不拖延；
    
- 每周日，筛选本周记录中的精华题，移动到「01_按题型分类」文件夹；
    
- 可快速回顾自己的刷题进度，避免盲目刷题、重复刷题。
