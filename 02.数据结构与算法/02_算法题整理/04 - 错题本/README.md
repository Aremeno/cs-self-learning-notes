## 文件夹用途

存放所有「反复错、卡壳、看题解才会」的题目，核心作用是「针对性突破薄弱点」，避免重复踩坑，高效提升。

## 内容规范

1. 两类内容：单道错题笔记 + 错误类型总结；
    
2. 单道错题笔记：单独建 .md 文件，文件名格式：「题号-题目名-错误类型.md」（例：LeetCode3-无重复字符的最长子串-边界错误.md），内容包含：题号&题目、错误原因、正确思路、核心代码片段、下次避免方法；
    
3. 错误类型总结：单独建「错误类型合集.md」，按错误类型分类（例：边界错误、DP状态定义错误、输入输出错误），记录同类错误的共性问题；
    
4. 仅存放「真正值得反复看」的错题，偶然失误、笔误的题目不放入。
    
## 使用建议

- 刷题时做错的题，当天整理到此处，明确错误原因，不遗漏；
    
- 每周复习1次错题，重做一遍，检验是否真正掌握；
    
- 错误类型合集定期更新，总结共性问题，避免重复犯错。
